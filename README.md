# Security Notes for Beginners
Simple, practical, core-focused security notes from basic to advanced.
For learning, reviewing, and building real security skills.

## Topics
- Cybersecurity Fundamentals
- Network Security
- System Security
- Web Security (OWASP Top 10)
- Password & Hash Security
- Security Tools (Nmap, Wireshark, OpenSSL)
- Secure Coding & Best Practices

## Goal
- Understand core concepts clearly
- Learn how attacks work
- Know how to defend systems
- Be able to apply in real projects
