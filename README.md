# Security Notes
Simple, practical, core-focused cybersecurity notes from basic to advanced. Understand principles, and apply skills in real scenarios. 

All notes are clear theory + practical examples + security defenses, focusing on core knowledge.

## 📚 Module Overview (All Modules)
This repository covers 7 core cybersecurity modules, from fundamentals to best practices.

### 01. Cybersecurity Fundamentals (基础入门)
Core: Understand the basics of cybersecurity, key concepts, and common threats.
- 01-What-is-Cybersecurity.md - Definition, importance, and basic practices
- 02-CIA-Triad.md - The foundation of cybersecurity (Confidentiality, Integrity, Availability)
- 03-Common-Threats-Vulnerabilities-Exploits.md - Differentiate threats, vulnerabilities, and exploits
- 04-Attack-Types-Overview.md - Common attack types (Phishing, Malware, MitM, Brute Force)

### 02. Network Security (网络安全)
Core: Learn network protocols, defense tools, and DDoS protection.
- 01-IP-TCP-UDP-Basics.md - Fundamental network protocols (IP, TCP, UDP)
- 02-Firewalls-101.md - Firewall basics, rules, and best configurations
- 03-IDS-IPS.md - Intrusion Detection/Prevention Systems (difference and usage)
- 04-DDoS-Basics-and-Protection.md - DDoS attacks and practical defense methods

### 03. System Security (系统安全)
Core: Secure operating systems, user access, and patch management.
- 01-OS-Permissions-Basics.md - OS permission management (least privilege principle)
- 02-User-Authentication.md - Authentication methods (password, 2FA, MFA)
- 03-Patch-Management.md - Patch importance and deployment best practices

### 04. Web Security (Web安全)
Core: OWASP Top 10, common web vulnerabilities, and prevention methods.
- 01-HTTP-HTTPS.md - Difference between HTTP and HTTPS, secure transmission
- 02-OWASP-Top-10.md - The most critical web security risks
- 03-XSS-Attack-Prevention.md - XSS attacks and defense measures
- 04-SQL-Injection-Prevention.md - SQLi attacks and secure coding practices
- 05-CSRF-Prevention.md - CSRF attacks and protection methods

### 05. Password Security (密码安全)
Core: Secure password storage, hashing, and policy.
- 01-Hash-vs-Encrypt.md - Difference between hashing and encryption (critical for passwords)
- 02-Strong-Password-Policy.md - Rules for creating strong, secure passwords
- 03-Salted-Hashes.md - Salted hashes to prevent rainbow table attacks

### 06. Security Tools (安全工具)
Core: Practical use of common security tools (beginner-friendly).
- 01-Nmap.md - Network scanning, host/port detection
- 02-Wireshark.md - Network packet analysis and traffic monitoring
- 03-OpenSSL.md - TLS/SSL encryption, certificate management

### 07. Best Practices (最佳实践)
Core: Apply cybersecurity knowledge to real scenarios.
- 01-Secure-Coding-Guide.md - Secure coding practices to prevent vulnerabilities
- 02-Zero-Trust-Basics.md - Zero Trust framework (modern security approach)

## 📌 How to Use This Repository
1. Learn in the recommended order (beginner path above).
2. Read each note carefully, focus on "Core Concept" and "Key Takeaways" for quick review.
3. Practice the "Practical Example" section to apply what you learn.
4. Use the "How to Defend / Secure" section to implement security measures in real projects.

Happy learning! This repository is designed to be your go-to reference for cybersecurity basics — come back anytime to review and refresh your knowledge.
